## Installation

### Download AppImage

1. Go to [Releases](../../releases)
2. Download `RommSync-x86_64.AppImage`
3. Copy to Steam Deck
4. Make executable: `chmod +x RommSync-x86_64.AppImage`
5. Run: `./RommSync-x86_64.AppImage`
6. Open browser to `http://localhost:5000`
